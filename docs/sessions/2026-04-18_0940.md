   pip recusou o upgrade; precisou
   `--ignore-installed typing_extensions` primeiro. Adicionar ao
   installer (`[0/7]` pip install com verificação prévia).

   5 tentativas falhadas com deps ausentes, systemd rejeitou
   `restart`. Joao teve que `systemctl reset-failed`. Comportamento
   CORRETO — o limit fez o que deve. Mas o installer deveria rodar
   `reset-failed` antes de `restart` por garantia (barato).
3. **`started_at` igual a `last_tick_at`** no run antigo (2026-04-18_0229)
   porque foi iniciado ANTES do manifest writer (Task 2). Runs novos
   vão ter manifest correto. Cosmético, self-fixing.
