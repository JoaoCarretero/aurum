## Achados
- **Paralelismo silencioso**: `PaperclipClient` era acessado por 3 threads distintas (main poll + live runs modal + cost dashboard) sem lock no circuit breaker. Reviewer round-2 pegou isso — `_consecutive_failures` + `_breaker_open_until` podiam corromper em race. Fix: `threading.Lock`.
- **Sobrescrita de persona**: double-click em EDIT PERSONA abria 2 editores no mesmo arquivo. Segundo save overwrite o primeiro sem warning. Fix: ref + `winfo_exists()` guard.
- **Memory leak em `_tracked_after_ids`** (round-1): Screen reused + `_poll_state` reagendando a si mesma a cada 5s acumulava ~720 ids/hora com closures presas. Fix: `_after` wrappa callback e auto-poda id da lista quando dispara.
- **`_poll_state` sync HTTP bloqueava Tk** (round-1): is_online + list_agents + list_issues na main loop -> freeze até 1.5s/tick. Fix: thread daemon + `container.after(0, ...)` pra resultado.
- **`normalize_stem` perdia engines em titulos com brackets** (round-2): `[PHI] fib` virava `[phi]-fib` — prefix match de detect_engine falhava silent. Fix: pre-pass `[^a-z0-9\-]+`.
